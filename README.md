# THE DINER - Online Food Ordering System

A fully responsive and interactive web application for a fast-food restaurant, built with pure HTML, CSS, and JavaScript. This project demonstrates a complete e-commerce flow from product browsing to order confirmation.

## Features
- **Dynamic Menu**: Displays food items with images, descriptions, and prices.
- **Add to Cart**: One-click add functionality with quantity controls (+/-).
- **Real-time Price Calculation**: Automatically updates total price in the cart.
- **LocalStorage Persistence**: Cart data is saved in the browser, so it persists even after page refresh.
- **Multi-Step Checkout**: 
  - Cart page with item list and total.
  - Checkout form with customer details (Name, Address, Phone, Email).
  - Order Confirmation page displaying full summary.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop.
- **Smooth Animations**: Integrated ScrollReveal for modern UI interactions.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (ES6+, DOM Manipulation, LocalStorage)
- ScrollReveal.js (for scroll animations)
- Remix Icons

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.

## Project Structure
- `index.html` - Main menu/landing page
- `addtocart.html` - Shopping cart page
- `checkout.html` - Customer details form
- `confirm.html` - Order confirmation summary
- `addtocart.css` - All styling (dark theme, neon accents)
- `addtocart.js` - Main logic (cart management, localStorage)
- `products.json` - Mock data for product items
- `images/` - All logo, food, and background assets
